Error Code References

This reference lists common error codes encountered during the setup and operation of IoT devices. Each entry includes the error code, system-generated message, a brief description, and a suggested resolution or cross-reference to the appropriate troubleshooting topic.

Error Code Table:

The following table outlines key error codes reported by the system. These codes help users identify and resolve issues such as connectivity failures, authentication problems, or firmware update interruptions. Where applicable, troubleshooting references are provided for further assistance.

Error Code Message Description Suggested Resolution
E101 Login Failed Incorrect credentials entered or authentication timeout. Verify username/password and try again.
E102 Network Timeout Device failed to communicate with server. Check internet connection or troubleshoot_connectivity.dita
E201 Sync Failure Device unable to sync settings with the cloud. Restart device and ensure internet is stable.
E301 Firmware Update Failed Installation interrupted or version mismatch. Ensure stable power/network.
E401 Device Not Found Hub failed to detect the IoT device. Re-scan or reset the device before retrying.
E501 Unsupported Protocol Connected device uses an unsupported communication protocol. Refer to supported_devices_matrix.dita for valid options.
E601 App Version Outdated Companion app version is not compatible with the device. Update app from official store and retry.
